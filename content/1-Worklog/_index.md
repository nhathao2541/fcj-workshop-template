# Worklog

During the **Workforce Bootcamp - First Cloud AI Journey** program, I completed the internship program within **12 weeks**, from **20/04/2026 to 10/07/2026**.

During the internship period, I learned and practiced fundamental knowledge of **Cloud Computing**, **Amazon Web Services (AWS)**, and the process of deploying applications on cloud platforms.

The internship content was divided into 12 weeks, including learning about AWS fundamentals, computing services, storage services, networking, databases, Serverless architecture, monitoring systems, and developing a Mini Project using AWS services.

The main activities completed during each week are as follows:

**Week 1:** [Introduction to AWS, Cloud Computing, and basic AWS services](1.1-week1/)

**Week 2:** [Learn about AWS IAM and Amazon VPC, practice access management and AWS network configuration](1.2-week2/)

**Week 3:** [Learn about AWS storage services including Amazon S3 and Amazon EBS](1.3-week3/)

**Week 4:** [Learn about Serverless architecture with AWS Lambda and Amazon API Gateway](1.4-week4/)

**Week 5:** [Learn about Amazon RDS database services and practice database management on AWS](1.5-week5/)

**Week 6:** [Deploy a static website using Amazon S3 Static Website Hosting and learn about Amazon CloudFront](1.6-week6/)

**Week 7:** [Deploy a website on Amazon EC2 using Apache/Nginx Web Server](1.7-week7/)

**Week 8:** [Learn about Docker Containers and practice deploying applications using Docker on EC2](1.8-week8/)

**Week 9:** [Learn about Amazon CloudWatch and practice monitoring AWS resources](1.9-week9/)

**Week 10:** [Start developing the Mini Project, design the system architecture, and prepare AWS resources](1.10-week10/)

**Week 11:** [Complete the Mini Project, test system functionalities, and optimize the system](1.11-week11/)

**Week 12:** [Summarize the internship process, complete the report, and prepare for the project presentation](1.12-week12/)
