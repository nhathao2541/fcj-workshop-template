### Week 2 Objectives

* Learn how to manage users and access permissions on AWS.
* Get familiar with Amazon VPC services.
* Understand how to connect EC2 instances to the Internet.
* Practice creating and configuring basic networking resources.

### Tasks Completed During the Week

| Day | Task | Start Date | Completion Date | Reference Materials |
| --- | --------- | ------------ | --------------- | -------------- |
| 2 | Learn about AWS IAM, create IAM Users, IAM Groups, and assign permissions using IAM Policies. Log in to AWS using the newly created IAM account. | 27/04/2026 | 27/04/2026 | https://cloudjourney.awsstudygroup.com/ |
| 3 | Learn about Amazon VPC, create a new VPC, configure CIDR Block, and create a Public Subnet. | 28/04/2026 | 28/04/2026 | https://cloudjourney.awsstudygroup.com/ |
| 4 | Create an Internet Gateway, configure Route Table, and associate it with the Public Subnet to enable Internet access. | 29/04/2026 | 29/04/2026 | https://cloudjourney.awsstudygroup.com/ |
| 5 | Create Security Groups, open SSH (22), HTTP (80), and HTTPS (443) ports. Launch an EC2 instance inside the created VPC and verify SSH connectivity. | 30/04/2026 | 30/04/2026 | https://cloudjourney.awsstudygroup.com/ |
| 6 | Review the learned knowledge, use AWS CLI to check information about IAM Users, EC2 Instances, and VPC resources. Document the implementation steps for learning materials. | 01/05/2026 | 01/05/2026 | https://cloudjourney.awsstudygroup.com/ |

### Week 2 Achievements

* Understand the role of AWS IAM in user management and access control.

* Successfully create IAM Users, IAM Groups, and assign permissions using IAM Policies.

* Successfully create an Amazon VPC with a Public Subnet.

* Configure Internet Gateway, Route Table, and Security Group to allow EC2 instances to access the Internet.

* Launch an EC2 instance inside the VPC and successfully connect using SSH.

* Learn how to use AWS CLI to check information about several AWS resources.

* Complete practical exercises on IAM and VPC, building a foundation for learning AWS storage services.
