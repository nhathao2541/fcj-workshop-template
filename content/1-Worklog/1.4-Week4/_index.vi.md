### Kết quả đạt được tuần 4

* Hiểu được cách hoạt động của mô hình Serverless trên AWS.

* Tạo và kiểm thử thành công AWS Lambda Function.

* Tạo được REST API bằng Amazon API Gateway và kết nối với Lambda.

* Kiểm tra được API bằng Postman và nhận dữ liệu trả về ở định dạng JSON.

* Biết sử dụng CloudWatch Logs để xem kết quả thực thi và xử lý lỗi cơ bản.

* Hoàn thành các bài thực hành về Lambda và API Gateway, chuẩn bị cho việc xây dựng ứng dụng AWS ở các tuần tiếp theo.
