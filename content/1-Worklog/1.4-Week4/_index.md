### Week 4 Objectives

* Learn about the Serverless model on AWS.
* Get familiar with AWS Lambda and Amazon API Gateway.
* Practice creating simple Lambda functions.
* Understand how Lambda and API Gateway work together.

### Week 4 Achievements

* Understand how the Serverless model works on AWS.

* Successfully create and test AWS Lambda Functions.

* Create a REST API using Amazon API Gateway and integrate it with Lambda.
