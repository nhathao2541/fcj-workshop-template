### Week 5 Objectives

* Learn about database services on AWS.
* Get familiar with Amazon RDS.
* Understand how to create and manage databases in the Cloud.
* Practice connecting applications with databases.

### Tasks Completed During the Week

| Day | Task | Start Date | Completion Date | Reference Materials |
| --- | --------- | ------------ | --------------- | -------------- |
| 2 | Learn about different types of databases on AWS and understand the differences between relational databases (RDS) and NoSQL databases (DynamoDB). | 18/05/2026 | 18/05/2026 | https://cloudjourney.awsstudygroup.com/ |
| 3 | Learn about Amazon RDS, supported Database Engines, and the process of creating an RDS Instance. | 19/05/2026 | 19/05/2026 | https://cloudjourney.awsstudygroup.com/ |
| 4 | Practice creating an Amazon RDS MySQL instance, configuring Security Groups, and connecting to the database using MySQL Workbench. | 20/05/2026 | 20/05/2026 | https://cloudjourney.awsstudygroup.com/ |

| 6 | Learn about Amazon RDS Backup, Snapshot, and Multi-AZ features. Review the knowledge learned during the week. | 22/05/2026 | 22/05/2026 | https://cloudjourney.awsstudygroup.com/ |

### Week 5 Achievements

* Understand the role of Amazon RDS in managing databases on AWS.

* Successfully create an Amazon RDS MySQL Instance and connect to the database.

* Perform basic SQL operations such as creating tables, inserting data, and querying data.

* Learn how to configure Security Groups to allow connections to RDS.

* Understand the functions of Backup and Snapshot for data backup and recovery.

* Complete practical exercises on Amazon RDS and gain fundamental knowledge for working with databases on AWS.
