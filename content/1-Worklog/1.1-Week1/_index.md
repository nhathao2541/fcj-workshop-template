### Week 1 Objectives

* Get familiar with the First Cloud AI Journey program.
* Learn the basic knowledge of AWS and Cloud Computing.
* Set up the environment for learning and practicing on AWS.
* Get familiar with basic AWS services.

### Week 1 Achievements

* Understand the basic concepts of Cloud Computing and AWS core services.

* Successfully create an AWS Free Tier account and receive AWS Credits for the learning process.

* Install and configure AWS CLI to connect and interact with AWS services.

* Become familiar with the AWS Management Console interface and learn how to search for AWS services.

* Successfully create an EC2 Instance and connect to the server using SSH.

* Understand the roles of AMI, Security Group, Key Pair, and Amazon EBS when deploying EC2 instances.

* Learn how to monitor AWS Credits and track usage costs through the AWS Billing Dashboard.
