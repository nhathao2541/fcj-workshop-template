### Week 3 Objectives

* Learn about storage services on AWS.
* Understand the differences between Amazon S3 and Amazon EBS.
* Practice storing and managing data on AWS.
* Get familiar with data backup features.

### Tasks Completed During the Week

| Day | Task | Start Date | Completion Date | Reference Materials |
| --- | --------- | ------------ | --------------- | -------------- |
| 2 | Learn about Amazon S3, including how to create Buckets, Bucket naming rules, and data management on S3. | 04/05/2026 | 04/05/2026 | https://cloudjourney.awsstudygroup.com/ |
| 3 | Practice creating an S3 Bucket, uploading and downloading files, creating folders, and modifying Bucket access permissions. | 05/05/2026 | 05/05/2026 | https://cloudjourney.awsstudygroup.com/ |
| 4 | Learn about S3 Versioning, Storage Classes, and Lifecycle Rules. Practice enabling Versioning and configuring Lifecycle Rules for a Bucket. | 06/05/2026 | 06/05/2026 | https://cloudjourney.awsstudygroup.com/ |

| 6 | Practice creating EBS Snapshots, restoring data from Snapshots, and review the storage services learned during the week. | 08/05/2026 | 08/05/2026 | https://cloudjourney.awsstudygroup.com/ |

### Week 3 Achievements

* Understand the functions and purposes of Amazon S3 and Amazon EBS.

* Successfully create and manage Amazon S3 Buckets for data storage.

* Practice uploading, downloading, and managing files on Amazon S3.

* Learn how to use Versioning and Lifecycle Rules for effective data management.

* Successfully create and attach an Amazon EBS Volume to an EC2 Instance.

* Perform EBS Snapshot operations to back up data.

* Complete practical exercises on AWS storage services and understand how to apply them in AWS environments.
